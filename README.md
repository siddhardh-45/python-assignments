# User Profile Validation in Python

This program validates a user's profile based on:

- Full name format
- Email structure
- Mobile number correctness
- Age range

## Conditions

Full Name:
- Must contain first and last name
- No leading or trailing spaces

Email:
- Must contain '@' and '.'
- Should not start with '@'

Mobile Number:
- Must be 10 digits
- Only numeric values
- Should not start with 0

Age:
- Must be between 19 and 60

## Output

Displays:
User Profile is VALID
or
User Profile is INVALID
